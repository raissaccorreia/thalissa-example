# thalissa-example

## DB

### Running DB

```bash
docker-compose up -d
```

### Stopping DB

```bash
docker-compose down
```

### Resetting DB

```bash
docker-compose down -v
```

## Running API

```bash
cd thalissa-nest
npm run start:dev
```

## Running UI

```bash
cd thalissa-remix
npm run dev
```
